# 本地数据分析与清洗平台

一个基于 Python 和 Streamlit 的本地单机 Web 应用，用于上传 CSV/Excel 文件，进行数据子集选择、数据处理、可视化分析和结果导出。

## 功能

- 上传 `CSV` 与 `XLSX` 文件，Excel 支持选择单个工作表
- 在表格中勾选要参与分析的列，并查看列名、数据类型、缺失值、唯一值等信息
- 按 `全部行`、`行号区间`、`手动行号`、`前N行`、`预览勾选` 选择当前分析子集
- 按文本、数值区间、日期区间、空值条件筛选数据
- 执行缺失值处理、删除重复值、类型转换、异常值处理、重命名列、删除列、排序、数值缩放
- 使用图表工坊生成柱状图、堆叠柱状图、折线图、面积图、散点图、箱线图、小提琴图、直方图、饼图、热力图
- 执行聚合分析、透视分析、时间趋势分析、文本高频词分析和词云分析
- 使用宽表分析直接处理 `科目名称 + 1月到12月` 这类业务表，自动生成多条线月趋势、分组对比和热力图
- 导出当前处理后数据、当前分析子集、当前图表，以及分析报告 HTML

## 快速开始

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

Windows 也可以直接运行：

```bash
start_app.bat
```

## 正确启动方式

这是一个 Streamlit 应用，不能使用 `python app.py` 直接启动。

错误示例：

```bash
python app.py
```

正确示例：

```bash
streamlit run app.py
```

如果你在虚拟环境中，推荐使用：

```bash
.\.venv\Scripts\streamlit run app.py
```

启动成功后，在浏览器打开：

```text
http://localhost:8501
```

## 核心使用流程

1. 上传 CSV 或 Excel 文件
2. 在“数据预览与选择”中勾选分析列，并选取当前分析子集
3. 如有需要，使用条件筛选和数据处理模块清洗数据
4. 在“可视化与分析”中选择普通图表分析、聚合分析、宽表分析、透视分析、时间趋势或文本分析
5. 在“导出结果”中下载处理后数据、分析子集、图表和分析报告

## 重点分析能力

### 1. 数据子集分析

- 可以同时选择多列，而不是只做单列分析
- 可以单独圈定某几行或某个行区间作为分析对象
- 所有分析模块都基于你当前选中的“分析子数据框”运行

### 2. 宽表分析

适合以下结构的数据：

```text
科目名称 | 1月 | 2月 | 3月 | ... | 12月
工资     | ... | ... | ... |
差旅费   | ... | ... | ... |
福利费   | ... | ... | ... |
```

支持：

- 选择一个标签列，例如 `科目名称`
- 选择多个值列，例如 `1月` 到 `12月`
- 选择多个行标签，例如 `工资`、`差旅费`、`福利费`
- 自动转成长表后生成：
  - 多条线趋势图
  - 分组柱状图
  - 堆叠柱状图
  - 面积图
  - 对比热力图

### 3. 透视分析

- 通过行维度、列维度和值列生成透视表
- 自动展示热力图
- 已处理重复选同一列导致的 `Grouper not 1-dimensional` 报错

### 4. 分析报告导出

可导出 `analysis_report.html`，报告包含：

- 数据来源
- 分析范围
- 筛选条件
- 处理历史
- 缺失值统计
- 描述性统计
- 当前图表
